What is Quantum Computing?

Quantum computing is a new paradigm of computing which uses physical quantum phenomena to drastically speed up certain computations. Unlike classical computers that rely on bits, which can be either 0 or 1, to process data, quantum computers use qubits which can be any value between 0 and 1, as well as complex values. This allows for much faster computations on problems such as factoring large numbers, which is a key part of modern encryption.
